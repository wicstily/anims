Анимации

Я подразделяю анимации на два вида: запускающиеся сразу и запускающиеся с каким-то эвентом

Пример запускающихся сразу

Такие анимации создаются с помощью css

Вот код этого примера: 

          
.a-start {  
  animation: spin 7s linear infinite;
}

@keyframes spin {
  from { transform:rotate(0deg) }
  to { transform:rotate(360deg) }
}

То есть, создаётся элемент с шириной и высотой, в css ему прописывается свойство Animation, указывается имя анимации (spin), время выполнения в секундах или миллисекундах (7s), функция выполнения (linear), количество проходов анимации (infinite).

Далее прописывается правило @keyframes, в котором указывается то, что будет анимировано. В моём случае это просто вращение, поэтому там прописаны лишь свойства, вращающие элемент. Здесь можно указывать или два ключевых слова from to и прописывать анимацию в них, или указывать процентами от 0% до 100%.

Примеры запуска по эвенту

Привет, я скрытый текст. А ещё во мне есть вот это: Lorem ipsum dolor sit amet consectetur adipisicing elit. Dignissimos officia facilis blanditiis architecto quod incidunt reprehenderit asperiores necessitatibus explicabo dicta, ea ad cumque harum nulla voluptatem accusamus quas culpa fugit!

Это делается с помощью js и может быть сделано двумя способами: прописыванием стилей и добавлением класса. Оба примера есть выше.

Первый

css:

#btn {
  transition-property: background-color;
  transition-duration: 1.5s;
}

js:

let btn = document.querySelector('#btn')

btn.addEventListener('click', () => {
  btn.style.backgroundColor = 'black'
})

Второй

css:

.hidden {
  height: 0px;
  opacity: 0;
  transition: all 1.5s ease-in-out;
}

.show {
  opacity: 1;
  height: 100px;
}

js:

let btn2 = document.querySelector('#btn2')

btn2.addEventListener('click', () => {
  hp.classList.add('show')
})

В первом случае мы добавляем элементу свойство transition для цвета фона и меняем цвет с помощью скрипта

Во втором же мы создаём отдельный класс show и добавляем его к элементу при нажатии кнопки

Стили создания анимации можно смешивать между собой. Например, можно создать класс, у которого прописана анимация с помощью @keyframes и добавлять этот класс по нажатию кнопки.
